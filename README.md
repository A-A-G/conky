* weather (including 9-day forecast)

* rss (arch news, subcentral and spon)
* bitcoin values from Coinbase.com and Bitcoin.de (for Bitcoin.de Imagemagick is used)
* altcoins (hitbtc.com): XRP, TRX, BCH, ETH
